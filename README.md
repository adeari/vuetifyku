# vueku

## Project setup
```
yarn install
```

Setting ini 
```
Rename vue.config.js_example menjadi vue.config.js
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
